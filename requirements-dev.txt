# Sysadmin Security Lab - Developer Workstation Requirements

#

# Purpose

# -------

# This file installs the core dependencies required to develop,

# maintain, test, document, and operate the Sysadmin Security Lab.

#

# Included Categories

# -------------------

# - Infrastructure Automation (Ansible)

# - DevOps & Infrastructure as Code (IaC)

# - Security Engineering & Validation

# - Python Development & Testing

# - Documentation Generation

# - General Utilities

#

# Notes

# -----

# Runtime dependencies for individual lab environments are maintained

# separately within their respective directories:

#

# labs/security/*/requirements.txt

#

# Example Usage

# -------------

# Create and activate a virtual environment:

#

# python3 -m venv .venv

# source .venv/bin/activate

#

# Install dependencies:

#

# pip install --upgrade pip

# pip install -r requirements.txt

# Tutorials & Reference Guides

This directory contains **educational tutorials, guides, and reference material** to support security, system administration, and PJPT-style Active Directory labs.

All tutorials are intended for **learning and authorized practice only**. They complement the scripts and labs in this repository.

---

## Contents Overview

### Active Directory & Security

- **AD_MITRE_log_source_playbook.md** — Mapping MITRE ATT&CK log sources for AD environments
- **Windows Server 2025 Active Directory Assessment.md** — Assessment and configuration guide for AD labs
- **Pentest Tools.md** — Reference list of essential security tools for Linux and Windows

### Linux & System Administration

- **Apparmor.md** — Linux AppArmor security guide
- **Complete Git Tutorial for Linux Users.md** — Step-by-step Git workflow for sysadmins
- **Complete Pacstall Tutorial The AUR for Ubuntu.md** — Pacstall package manager usage
- **KVM-QEMU Installation.md** — KVM/QEMU hypervisor installation
- **Managing KVM QEMU from the Command Line (CLI) - A Complete Tutorial.md** — CLI-based management of VMs
- **QEMU and KVM CLI Management Tutorial.md** — VM management using QEMU/KVM CLI
- **TimeShift CLI Guide (Including External Drive Backups).md** — Backup and restore using TimeShift
- **Vagrant Management Tutorial and Setup Guide.md** — Vagrant usage and environment setup

## How to Use

1. Browse the tutorials folder to select the guide relevant to your task or lab.
2. Follow the step-by-step instructions in each Markdown file.
3. Use the tutorials in conjunction with scripts and labs for **practical, hands-on learning**.

> Note: All tutorials are designed for **authorized environments**. Do not attempt on production systems without permission.

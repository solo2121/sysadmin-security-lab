kept for continuity — general Linux/sysadmin tutorials written before the
current lab structure existed. They are **not** actively maintained
against the current labs and may reference tools, versions, or workflows
that have since changed.

If you're looking for current, maintained documentation for the labs
themselves, start at [`docs/README.md`](../README.md) instead.
